ALU ​

ALU（Arithmetic Logic Unit）算术逻辑单元

4-bit ALU 可以进行0+000：1位表示符号3位表示二进制数值运算

Operator Code（opcode）三位000，设计5种操作

操作码OP功能
000A ADD B
001A SUB B
010A AND B
011A OR B
100A XOR B

补码（Two’s Complement） ​

补码的意义就是用加法实现减法：将要减的数取负数在进行加法

  • 7-5 —> 7 + (-5)

补码在 4-bit ALU 中的解析机制

  • 正数 = 原样解释
  • 负数 = 反码 + 1 后再加符号
 0111  (+7)
+1011  (-5) 101取反010+1 = 011 加上最左侧位bit表示符号
------
10010 → 截断 4-bit = 0010 = 2
